

本项目基于 MATLAB 实现了电场、磁场典型模型的可视化仿真，涵盖导体球 / 介质球外电场、有限长带电棒电场、无限长直导线磁场、亥姆霍兹线圈磁场等经典电磁学场景，可直接运行代码生成场分布的等势面 / 磁感线、矢量箭头等可视化结果。

## 项目结构

- **模拟磁场/** <!-- markdownlint-disable MD037 -->
  - `current_loop.m` <!-- markdownlint-disable MD037 --> - 圆电流环磁场模拟
  - `double_wires.m` - 双平行直导线磁场模拟
  - `finite_solenoid.m` - 有限长螺线管磁场模拟
  - `helmholtz_coils.m` - 亥姆霍兹线圈磁场模拟
  - `long_wire.m` - 无限长直导线磁场模拟
  - `[PNG图片]` - 各模型的磁场分布可视化结果
- **模拟电场/**
  - `conducting_sphere.m` - 导体球在均匀外电场中的电场模拟
  - `dielectric_sphere.m` - 介质球在均匀外电场中的电场模拟
  - `finite_charged_rod.m` - 有限长带电细棒电场模拟
  - `[PNG图片]` - 各模型的电场分布可视化结果

## 运行说明
### ✅ 环境依赖
- 需安装 MATLAB（R2018b 及以上版本均可）

### ✅ 使用方法
1. 进入对应功能文件夹（如`模拟磁场/`、`模拟电场/`）；
2. 在 MATLAB 中直接打开目标`.m`文件，点击编辑器的「运行」按钮；
3. 程序自动执行计算并弹出场分布可视化窗口；
4. 代码内置全套参数配置（导体球半径、电流大小、带电棒长度等），直接修改代码内参数值，重新运行即可调整仿真条件。
